## 🗂 폴더 구조

각 스터디원은 **본인 이름 폴더** 안에서만 작업합니다.  
폴더 안에서 플랫폼별로 구분하고, 파일명과 폴더 구조는 아래 예시를 따르지 않고 자유롭게 하셔도 됩니다. 문제 번호와 제목은 알아볼 수 있게 해주세요.

```
algo-class4-group4/
├── README.md
├── .gitignore
│
├── Hyunwoo/
│   ├── boj/
│   │   └── 1260_DFS와BFS.py
│   └── programmers/
│       └── 오픈채팅방.py
│
└── Yoonsuh/
    └── boj/
        └── 1753_최단경로.py
```

> 파일명은 자유 — 단, 어떤 문제인지 알아볼 수 있게만 해주세요.

## ⚙️ 운영 방식

```
각자 브랜치에서 작업 후 Main으로 PR 날리기
```

1. `main` 브랜치에서 **본인 이름으로 브랜치** 생성
   ```bash
   git checkout -b 정윤서
   ```
2. 본인 폴더 안에 풀이 파일 추가
3. 커밋 & 푸시 후 **`main`으로 PR** 생성
